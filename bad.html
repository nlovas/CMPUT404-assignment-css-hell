About

My Work


ShareMyChart

ShareMyChart is a web application created with a team in CMPUT 401 for a client. This application takes Turtle file data from the user and generates graphs. View ShareMyChart Javascript (D3 and JQuery), SPARQL, HTML, and CSS.



PhotoSynthesis

PhotoSynthesis is a web application created with a team in CMPUT 391. PhotoSynthesis hosts images for its users similarly to Flickr or Photobucket. Users can upload photos, join groups, and search images. Admins can generate statistics on the users of PhotoSynthesis. HTML, CSS, Java & JDBC.



Bookwrm

Bookwrm is an android application created with a team in CMPUT 301. This application allows users to trade books with other users. It has many features including sending friend requests, photographing books, search, and viewable trade histories. Android & Java.
