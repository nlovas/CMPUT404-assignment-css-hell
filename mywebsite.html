My Work

ShareMyChart

December 2016. Github (unavailable)

ShareMyChart is a web application created with a team in CMPUT 401 for a client. This application takes Turtle file data from the user and generates graphs using Javascript (D3 and Jquery), SPARQL, HTML, and CSS. View ShareMyChart

PhotoSynthesis

April 2016. Github

PhotoSynthesis is a web application created with a team in CMPUT 391. PhotoSynthesis hosts images for its users similarly to Flickr or Photobucket. Users can upload photos, join groups, and search images. Admins can generate statistics on the users of PhotoSynthesis.

Bookwrm

December 2015. Github

Bookwrm is an android application created with a team in CMPUT 301. This application allows users to trade books with other users. It has many features including sending friend requests, photographing books, search, and viewable trade histories.
